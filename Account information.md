# Account information

**Path:** /ip/open-api/account-info

**Method:** GET

**Interface description: Obtain account information**

**Return data:**

```javascript
{
  "data": {
    "balance": "99945.09",
    "cashback": "0.00",
    "total_balance": "99945.09"
  },
  "code": 0,
  "msg": "Success"
}
```

 

Field description:

| Field Name    | Description           |
| ------------- | --------------------- |
| balance       | Balance               |
| cashback      | Bonus                 |
| total_balance | Total account balance |
